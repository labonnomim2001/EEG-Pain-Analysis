# EEG-Pain-Analysis

This repository contains all preprocessing scripts, feature extraction code, and machine learning experiments for my Chronic Neuropathic Pain EEG project.

## Contents
- `clean_eeg.py` — preprocessing pipeline (filtering, ICA, interpolation)
- `pain_ml.py` — PSD feature extraction + SVM/Random Forest classification
- Cleaned EEG files (.fif)
- Project reports and documentation

## Dataset
Dataset used: **Chronic Neuropathic Pain EEG Raw Data (EC/EO)**  
Mendeley DOI: https://doi.org/10.17632/yj52xrfgtz.1  

## Project Summary
This project explores:
- EEG preprocessing
- Power spectral density (PSD) extraction
- Artificial EC/EO labeling for ML practice
- SVM + Random Forest classification
